This common folder I repurposed from Vincent's "shared_assets", but they are quite different.
The point of this folder is to keep in mind that the Godot engine works just fine with 0 folders and all files,
so the structure is just there to help us. If ever there is something that seems to defy structure and categorization,
just chuck it in here. Now, will this folder itself be that flat structureless sea of files? No, there can be basic
structure in here, very few things are truly unstructurable, but this folder is both for those things
that can't be structured, and those that it is not worth the effort to scructure. If someone took the time, it is likely
all the things that would end up in this folder could go in their own like units, spells, screens, etc. and if they are
needed everywhere that's fine, the folder structure in no way restricts access or creates boundaries.

In short, this folder exists to save time, and if it grows too big, then that serves as an indicator
that some things need more thought in how they are structured.
