# MiniProject 2: Training and Evaluation Practices

The project uses various libraries, make sure to download them with the following command:
   ```pip install numpy pandas matplotlib scikit-learn```

To run the project, please make sure to un-comment the specific sections indicated in main.py. To uncomment blocks of code, select each line you want to run, and press `CTRL + /`.

In Task #2, only uncomment one of the trials at a time, and run the code. 

To run the file, run the following command after modifications **when CD'ed into the directory**:
   ```python main.py```

The dataset can be found here: https://archive.ics.uci.edu/dataset/94/spambase
